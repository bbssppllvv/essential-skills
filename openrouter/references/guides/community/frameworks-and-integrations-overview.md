# Frameworks and Integrations Overview

OpenRouter provides seamless compatibility with popular AI frameworks and SDKs, allowing developers to integrate advanced language models into their applications.

## Primary Framework Integrations

The documentation highlights eight main framework options:

- **Effect AI SDK** - For TypeScript Effect applications
- **LangChain** - Python and JavaScript support
- **LlamaIndex** - RAG applications in Python and TypeScript
- **Mastra** - Unified AI model access interface
- **OpenAI SDK** - Official SDK for Python and TypeScript
- **PydanticAI** - Python application development
- **TanStack AI** - React, Solid, and Preact frameworks
- **Vercel AI SDK** - Next.js application integration

## Additional Tool Integrations

Beyond primary frameworks, OpenRouter supports integration with several coding assistants and observability tools:

- Aider, Cline, Kilo Code, and Roo Code coding assistants
- VSCode Copilot and Xcode development environments
- Langfuse for observability and tracing capabilities

The documentation references a GitHub repository containing additional implementation examples for developers seeking more detailed guidance on specific integrations.
