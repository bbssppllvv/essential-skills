# OpenRouter & Zapier Integration Documentation

## Overview

OpenRouter provides integration with Zapier, enabling users to connect over 300 AI models to 8,000+ applications for automated workflows without coding requirements.

## Key Capabilities

The integration supports several advanced features:

- **Content Generation**: Access models including GPT-4, Claude, and Gemini
- **Data Analysis**: Leverage specialized models for domain-specific tasks
- **Image Processing**: Utilize vision-capable models for visual content
- **Structured Outputs**: Implement JSON mode and function calling
- **Real-time Streaming**: Enable response streaming for dynamic applications

## Setup and Features

Users can explore available automations and create their first workflow by accessing the Zapier Elements embedded on the integration page. The platform handles authentication, model selection, and error management automatically.

## Integration Benefits

As noted in the documentation: "Combine OpenRouter's model routing with Zapier's integrations to automate tasks across CRMs, spreadsheets, messaging, and more."
